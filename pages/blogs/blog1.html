Understanding Two Pointers

The two-pointer technique is one of the most common strategies in solving array and string problems efficiently. It involves using two indices that traverse the data structure from different positions...

When to Use?

  • Finding pairs that satisfy a condition
  • Removing duplicates in sorted arrays
  • Sliding window variations

Example Problem

function twoSum(nums, target) {
  let l = 0, r = nums.length - 1;
  while (l < r) {
    let sum = nums[l] + nums[r];
    if (sum === target) return [l, r];
    else if (sum < target) l++;
    else r--;
  }
}

This approach reduces the time complexity compared to brute force O(n²) to O(n) in sorted arrays.
